Assumptions:
* Using Centos 6.4
* Have and know how to use github account
* You have EPEL repos installed on the system and they are enabled
  wget
http://dl.fedoraproject.org/pub/epel/6/x86_64/epel-release-6-8.noarch.rpm
  sudo rpm -ivh epel-release-6-8.noarch.rpm
* git is installed


Steps To Setup Django Development Environment
1) Install pip -> sudo yum install python-pip
2) Install virtualenv -> sudo pip install virtualenvwrapper
3) Setup virtual env
  A) Add the following to your .basrhc file:
    export WORKON_HOME=$HOME/.virtualenvs
    export PROJECT_HOME=/directory/you/do/work #i recommend /data/threeandout
    source /usr/bin/virtualenvwrapper.sh
  B) Source your .basrhc file
  C) Create your virtual environment -> mkvirtualenv ENVIRONMENT_NAME
(whatever you want, say, threeandout)
  D) You type deactivate to leave
  E) To go into an environment, type 'workon ENVIRONMENT_NAME'
4) Clone the threeandout repo -> git clone
http://github.com/jxp360/threeandout.git
5) Checkout the develop branch, git checkout develop in the
/data/threeandout/threeandout directory
5) Install yum install python-devel
6) While in the virtual environment, install all dev dependecies -> pip
install -r /path/to/requirements/dev.txt
7) Go to root of threeand out and syncdb ->
python manage.py syncdb --settings=threeandout.settings.dev (this assumues you have aa data directory already made for the database: /data/threeandout/dbs/)
8) Run the server ->
python manager.py runserver localhost:8000 --settings=threeandout.settings.dev
